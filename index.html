Coloring book

  1. Выбирайте любую картинку на фоне страницы левой кнопкой мыши и раскрашивайте.
  2. Можете нарисовать и раскрасить свою картинку на чистом холсте, используя инструмент Рисование линий.
    Предварительно щёлкните по кнопке Очистить для очистки холста.
    Для заливки используйте инструмент Заливка.
  3. Выбрать цвет по образцу можно щелчком по выбранному цвету на образце с нажатой левой или средней кнопкой мыши (или использовать пипетку).
  4. Нужные цвета выбираюйте на палитре левой кнопкой мыши.
  5. Можно раскрашивать каждую область отдельно по клику мыши или группу областей одновременно, проводя по ним курсором с нажатой левой кнопкой мыши.
  6. Для сохранения картинки щёлкните правой кнопкой мыши на холсте и выберите Сохраниь картинку как...
  7. Отменить одно последнее действие можно комбинацией клавиш Ctrl + Z.
  8. Для увеличения изображения на холсте пользуйтесь колёсиком мыши с нажатой клавишей "Ctrl".
  9. Ниже находится много тематических подборок картинок для раскраски. Подборки раскрываютя и свёртываются щелчком мыши по названию подборки.
