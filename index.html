РАСКРАСКА КАРТИНОК

  • Выбирайте любую картинку на фоне страницы левой кнопкой мыши и раскрашивайте.
  • Можно раскрашивать каждую область отдельно по клику мыши или группу областей одновременно, проводя по ним курсором с нажатой левой кнопкой мыши.
  • Для того, чтобы нарисовать свой рисунок или добавить своё изображение к вставленному включите режим Рисование линий.
  • Для сохранения картинки щёлкните правой кнопкой мыши на холсте и выберите Сохраниь картинку как...
  • Нужные цвета выбираюйте на палитре левой кнопкой мыши.
  • Для выбора цвета по образцу используйте пипетку (или щелчок по выбранному цвету на изображении с нажатой средней кнопкой мыши).
    Цвет с образца можно взять, кроме того, левой кнопкой мыши.
  • Можно закрасить с помощью кнопки Закрасить линии разметки лишние линии (это необходимо в основном для сложных раскрасок по номерам).
    Закрашенные линии разметки восстановить нельзя.
  • Отменить одно последнее действие можно комбинацией клавиш Ctrl + Z.
  • Для увеличения изображения на холсте пользуйтесь колёсиком мыши с нажатой клавишей "Ctrl".
  • Ниже находится много тематических подборок картинок для раскраски. Подборки раскрываютя и свёртываются щелчком мыши по названию подборки.
